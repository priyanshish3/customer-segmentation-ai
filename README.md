# 🤖 Customer Segmentation AI

A Machine Learning based Customer Segmentation Web Application that predicts customer segments using the K-Means Clustering algorithm. The application helps businesses understand customer behavior based on Age, Annual Income, and Spending Score.

---

## 🌐 Live Demo

🔗 https://customer-segmentation-ai-1.onrender.com

---

# 📌 Features

- 🔐 User Authentication (Signup/Login)
- 🤖 Customer Segment Prediction
- 📊 Interactive Analytics Dashboard
- 📈 K-Means Cluster Visualization
- 📉 Customer Count by Segment (Bar Chart)
- 🫧 Income vs Spending Score (Bubble Chart)
- 📝 Prediction History
- 📄 PDF Report Generation
- 📱 Responsive User Interface
- ☁️ Live Deployment using Render

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/priyanshish3/customer-segmentation-ai.git
```

Move into the project folder

```bash
cd customer-segmentation-ai
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```

Open your browser

```text
http://127.0.0.1:5000
```
