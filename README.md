# 🚀 AI Powered Customer Segmentation

A Machine Learning web application that predicts customer segments based on customer age, annual income, and spending score using the K-Means Clustering algorithm.

## 🌐 Live Demo

🔗 https://priyanshi03.pythonanywhere.com

---

## 📌 Features

- Predict customer segment instantly
- K-Means Clustering model
- Beautiful and responsive UI
- Customer insights dashboard
- Opportunity score
- Marketing recommendations
- Customer distribution graph

---
